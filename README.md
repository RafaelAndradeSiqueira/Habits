# Habits

Um aplicativo simples para monitoramento de hábitos — focado em produtividade, fácil de usar e de entender.

✨ Funcionalidades principais
- Criar, editar e remover hábitos
- Marcar hábitos como concluídos por dia
- Visualização rápida do progresso diário/semanal
- Autenticação e Autorização por usuário
- Sistema de Login

## Tecnologias
- Vitejs, Laravel, Docker e MySQL.

## Instalação
1. Clone o repositório:
   git clone https://github.com/RafaelAndradeSiqueira/Habits.git
2. Entre na pasta do projeto:
   cd Habits
3. Instale dependências:
   npm install
4. Rode em modo de desenvolvimento:
   npm run dev
5. Rode php artisan serve em outro terminal.


## Uso
- Abra http://localhost:8000 (ou porta configurada) após executar o servidor.
- Crie um hábito, defina a frequência e marque os dias conforme for cumprindo.
